# Skills

Guided, repeatable recipes for extending OrbiDICOM. Each lives under `.claude/skills/<name>/`
and is usable both by humans and by coding agents (Claude Code, or `orbidicom ai`).

| Skill                                                          | Use it to…                                               |
| -------------------------------------------------------------- | -------------------------------------------------------- |
| [add-a-tool](.claude/skills/add-a-tool/SKILL.md)               | Add a new measurement/manipulation tool + toolbar button |
| [add-a-data-source](.claude/skills/add-a-data-source/SKILL.md) | Connect a new PACS/backend by implementing `DataSource`  |
| [create-a-theme](.claude/skills/create-a-theme/SKILL.md)       | Restyle the viewer with theme tokens                     |
| [add-a-locale](.claude/skills/add-a-locale/SKILL.md)           | Translate the UI into a new language                     |
| [scaffold-an-app](.claude/skills/scaffold-an-app/SKILL.md)     | Create a configured viewer app from the template         |
| [propose-a-feature](.claude/skills/propose-a-feature/SKILL.md) | Turn an idea into a well-formed feature proposal/issue   |
