# Propose a feature

1. Check `ROADMAP.md` — is it already planned or explicitly out of scope?
2. Draft: the problem (what's hard today), a proposed solution, and scope/non-goals.
3. Open a GitHub issue with the feature template (or run `orbidicom ai` to draft it from a
   one-line description).
4. Tag it `proposal`; expect discussion before adoption.
