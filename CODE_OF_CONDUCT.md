# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

We are committed to a welcoming, harassment-free community. Unacceptable behavior may be
reported to the maintainers at **conduct@orbidicom.dev**. Reports are handled confidentially.

The full text of the Contributor Covenant v2.1 applies; see the link above.
